```
/lore add [参数]  修改物品lore
/lore set <行数> <内容> 更改X行的描述
/lore insert <行数> <内容> 强行插入X行添加描述
/lore delete <行数> 删除X行内容
/lore clear 清除所有描述以及物品属性

/lore name [参数] 修改物品名字

* /lore owner [参数] 修改头颅样式
```

有了以上指令你就可以做出一些rpg武器了
